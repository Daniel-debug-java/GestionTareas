# GestionTareas

Sistema basico de gestion de tareas para un equipo de trabajo, desarrollado en Java.

## Descripcion

Proyecto academico de la materia Estructura de Datos - Semana 7.
Implementa tres estructuras de datos fundamentales para organizar tareas de un equipo.

## Estructuras de datos utilizadas

- ArrayList: almacena el historico completo de todas las tareas registradas.
- Queue Cola: gestiona las tareas pendientes en orden de llegada FIFO.
- Stack Pila: registra las tareas finalizadas mas recientemente LIFO.

## Funcionalidades

- Registrar nuevas tareas con validacion de datos
- Listar todas las tareas
- Marcar tareas como finalizadas
- Consultar tareas pendientes
- Mostrar ultimas tareas finalizadas
- Buscar tarea por nombre
